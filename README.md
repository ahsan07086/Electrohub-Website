Electrohub is an e-commerce website for online shopping of  electronic products such as laptop, smartphone, camera etc.
It offers high quality gadget, devices and accessories for daily life needs.
Key Technology:
Frontend Technology: HTML,CSS,Javascript<br>
Backend Technology: PHP <br>
Database: MySQL <br>
Framework: Laravel <br>
Contributions:
1. Developed an adaptable web application where customer can search their favourite product and order it.<br>
2. Divided the whole website into two sections: admin and customer.<br>
3. Admin can perform CRUD(Create,Read,Update,Delete) operation.
